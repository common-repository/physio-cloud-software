== Description ==

Physio Cloud Software Plugin integrates [Physio Cloud Software](https://physiocloudsoftware.com) to provide physiotherapy clinics an option for their patients to book appointments online.

== Frequently Asked Questions ==

= Do I need to have a Physio Cloud Software subscription =

Yes, a monthly or yearly subscription is required.

= Is this plugin free =

Yes, the plugin is given free.

= How much does the subscription cost =

Please refer to the [link here](https://physiocloudsoftware.com#price-plan)

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Physio Cloud Software Plugin` and click 'Install Now' or click on the `upload` link to upload `physio-cloud-software-plugin.zip`
4. Click on `Activate plugin
5. Get your API key from the [Portal](https://portal.physio.bse.com.cy)
